# BigCake MVP Clean Architecture

# Getting Started
## Step 1
Add it in your root build.gradle at the end of repositories:
<pre>
allprojects {
  repositories {
	...
	maven { url 'https://jitpack.io' }
  }
}
</pre>
## Step 2
Add the dependency
<pre>
dependencies {
  compile 'com.github.huynhminhkiet:BigCake:v1.0'
}
</pre>
